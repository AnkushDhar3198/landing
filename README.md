# NovaPulse Landing Page

A modern, interactive landing page built with pure HTML, CSS, and JavaScript.

## Live Structure
- Source files: `landing.html`, `landing.css`, `landing.js`
- GitHub Pages build folder: `docs/`

## Run locally
Open `landing.html` directly in your browser or use Live Server.

## Deploy via GitHub Pages
1. Push this repository to GitHub.
2. In GitHub repo settings, open **Pages**.
3. Set source to **Deploy from a branch**.
4. Branch: `main`, folder: `/docs`.
5. Save and wait for deployment.

Your site URL will be:
`https://<your-username>.github.io/<repo-name>/`
